# Boom Bap Machine

A classic MPC-style 4x4 drum pad application for making boom bap hip-hop beats. It features a retro 90s aesthetic with a vintage LCD screen, rugged interface, and responsive pads that light up when tapped.

## Features
- **4x4 Drum Pads**: The iconic 16-pad grid layout used in vintage grooveboxes and samplers.
- **Retro Aesthetic**: Gritty dark grey interfaces, retro LCD style display for BPM/Swing, and tactile drum pads.
- **Responsive Animations**: Pads press down and emit a red glow when tapped, providing satisfying visual feedback.
- **Cross-Platform**: Fully responsive layout that runs beautifully on Mobile, Web, and Desktop.

## Tech Stack
- Flutter
- Dart

## Setup
1. Clone the repository.
2. Run `flutter pub get` to fetch dependencies.
3. Run `flutter run` to launch the app on your preferred device.
